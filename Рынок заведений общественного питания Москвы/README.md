# Рынок заведений общественного питания в Москве

## Описание

Исследование рынка заведений общественного питания в Москве для дальнейшего открытия небольшого оригинального кафе (гостей должны обслуживать роботы).

В результате исследования определили соотношение видов объектов общественного питания по количеству, а также соотношение сетевых и несетевых заведений. Определили среднее количество посадочных мест для каждого вида объекта общественного питания. Нашли топ-10 улиц по количеству объектов общественного питания, а также число улиц с одним объектом общественного питания. Для улиц с большим количеством объектов общественного питания посмотрели распределение количества посадочных мест. Дали рекомендации по формату заведения.

<details>
  <summary>Данные</summary>

— название объекта общественного питания;
— сетевой ресторан;
— тип объекта общественного питания;
— адрес;
— количество посадочных мест.

</details>
